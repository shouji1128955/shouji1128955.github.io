html基础

web应用程序：处理来自浏览器的请求响应的服务

浏览器本质是一个解释器

标签分为： 块级标签（block）

  1. 独占一行
  2. 可以设置长宽

h1-h6 p div ul li

内联标签（inline）

  1. 按内容占宽
  2. 不可以设置长宽

b strong a img input

嵌套原则： 块级标签可以嵌套内联 ，也可以嵌套块级（p标签除外） 内联嵌套内联

向服务器发送请求的形式：

  1. 浏览器的地址栏 GET

  2. a标签 GET

  3. form表单 GET/POST

  4. ajax请求 GET/POST js的技术

表单标签

<form action=""> 
    <p> 姓名： <input type="text"></p>
    <p> 密码:  <input type="password"></p>
    <p> 提交： <input type="submit"></p>
</form>

submit 点击提交之后，会打包form表单中的键值对，发送给浏览器，如果没有键，就会打包失败
默认请求方式就是get
get和post区别在于: 请求放在url中是get请求，如果放在请求体中，属于post请求
