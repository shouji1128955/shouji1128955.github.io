# NVIDIA Fabric Manager 安装指南

NVIDIA Fabric Manager 用于管理 NVLink / NVSwitch 拓扑，多 GPU 节点必须安装并启动该服务，否则 GPU 间通信会报错。

> **注意**：安装版本必须与已安装的 NVIDIA 驱动版本完全一致，否则服务无法启动。

---

## 安装 nvidia-fabricmanager

### CentOS / RHEL

```bash
# 指定驱动版本（与已装驱动保持一致）
version=520.60.13

# 添加 CUDA yum 源
yum -y install yum-utils
yum-config-manager --add-repo \
  https://developer.download.nvidia.com/compute/cuda/repos/rhel7/x86_64/cuda-rhel7.repo

# 安装 fabricmanager
yum install -y nvidia-fabric-manager-${version}-1
```

### Ubuntu

```bash
# 指定驱动版本（与已装驱动保持一致）
driver_version=525.60.13

# 提取主版本号（如 525）
driver_version_main=$(echo $driver_version | awk -F '.' '{print $1}')

# 下载 deb 包
wget https://developer.download.nvidia.com/compute/cuda/repos/ubuntu2004/x86_64/\
nvidia-fabricmanager-${driver_version_main}_${driver_version}-1_amd64.deb

# 安装
dpkg -i nvidia-fabricmanager-${driver_version_main}_${driver_version}-1_amd64.deb
```

---

## 启动 nvidia-fabricmanager 服务

```bash
# 重新加载 systemd 配置
systemctl daemon-reload

# 设置开机自启
systemctl enable nvidia-fabricmanager

# 启动服务
systemctl start nvidia-fabricmanager.service

# 查看服务状态
systemctl status nvidia-fabricmanager
```
